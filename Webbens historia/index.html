Webbens historia

World Wide Web (WWW) uppfanns av Tim Berners Lee i Mars 1989 när han jobbade för det europeiska forskningscentrumet CERN.

Men han gjorde det offentligt 30 April 1993, så att folk runt om i världen också skulle få använda WWW.

Tim skapade sin första hemsida (WWW) med hjälp av the line-mode browser simulator

https://line-mode.cern.ch/www/hypertext/WWW/TheProject.html

Under den tiden som han skapade sin sida så skapade han även HTML, URL, HTTP och DNS. HTML står för Hyper Text Markup Language och är det språk all webbsidor är uppbyggda av. URL står för Uniform Resource Locator som är addressen till en hemsida. HTTP står för Hypertext Transfer Protocol och kan användas för att överföra data. DNS står för Domain Name System som är som en telefonkatalog för webben.

Sveriges första hemsida

Sveriges första hemsida kom 1993 hos en datorförening i Linköpings universitet som heter Lysator. Lysator blev också även sjätte hemsidan i världen.

https://www.lysator.liu.se/

Innan WWW skapades så fanns det typ nästan inga system för att länka till saker.
